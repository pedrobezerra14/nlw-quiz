## 💻Como rodar este projeto?

Após clonar o repositório, acesse a pasta do projeto e execute os comandos abaixo:

```bash
npm install
npm run dev
```

Acesse [http://localhost:5173](http://localhost:5173) para visualizar a aplicação.
